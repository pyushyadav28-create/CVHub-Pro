# CVHub Pro - CV Management System

## 📌 Overview

CVHub Pro is a full-stack web application developed using **PHP**, **MySQL**, **Bootstrap**, and **XAMPP**. The system allows users to create and manage a professional CV through an easy-to-use dashboard.

This project was developed to strengthen my skills in PHP, MySQL, CRUD operations, authentication, session management, and responsive web development.
